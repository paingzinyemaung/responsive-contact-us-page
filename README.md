# ✉️ Modern Contact Us Page UI

ဒီ Project ဟာ **HTML5** နဲ့ **CSS3** ကိုသာ အသုံးပြုပြီး ဖန်တီးထားတဲ့ သန့်ရှင်းသေသပ်ပြီး ခေတ်မီတဲ့ **Contact Us Page** ဒီဇိုင်းတစ်ခုဖြစ်ပါတယ်။ User တွေနဲ့ ဆက်သွယ်ရလွယ်ကူစေမယ့် Layout ပုံစံကို အဓိကထား တည်ဆောက်ထားတာ ဖြစ်ပါတယ်။

---

## 🌟 Features (ထူးခြားချက်များ)

* **Dual-Column Layout:** ဘယ်ဘက်ခြမ်းတွင် ဆက်သွယ်ရန် လိပ်စာ၊ ဖုန်းနံပါတ်နှင့် အီးမေးလ်တို့ကို Icons များဖြင့် ဖော်ပြထားပြီး၊ ညာဘက်ခြမ်းတွင် Message ပေးပို့ရန် Form ကို စနစ်တကျ ထည့်သွင်းထားခြင်း။
* **Clean Typography:** ဖတ်ရလွယ်ကူပြီး မျက်စိပသာဒဖြစ်စေမယ့် စာလုံးပုံစံနှင့် အရောင်များကို အသုံးပြုထားခြင်း။
* **Interactive Elements:** "Send Now" button တွင် စိတ်ဝင်စားစရာကောင်းသော Hover effect ပါဝင်ခြင်း။
* **Responsive Design:** Screen အရွယ်အစားအမျိုးမျိုးတွင် ပုံစံမပျက်ဘဲ အဆင်ပြေစွာ မြင်တွေ့နိုင်ခြင်း။

## 🛠️ Built With (အသုံးပြုထားသော နည်းပညာများ)

* **HTML5** - စာမျက်နှာ၏ တည်ဆောက်ပုံ (Structure) နှင့် Form elements များအတွက်။
* **CSS3** - Layout (Flexbox), Styling နှင့် ခေတ်မီသော ဒီဇိုင်းပိုင်းများအတွက်။

---

## 🚀 How to Run (အသုံးပြုပုံ)

၁။ ဒီ Repository ကို **Clone** လုပ်ပါ သို့မဟုတ် **Download ZIP** ဆွဲပါ။
၂။ Folder ထဲရှိ `index.html` ဖိုင်ကို Browser ဖြင့် ဖွင့်လှစ်၍ ကြည့်ရှုနိုင်ပါသည်။
